# Estatemark — Static Marketing Site

Small static website for Estatemark (land-plot management). Pages are plain HTML and styles live in `css/estatemark_styles.css`.

Quick preview

- Open `index.html` in your browser to preview locally. No build step required.
- Alternatively, use VS Code Live Server extension and open the workspace root.

Notes for contributors

- This repo is served via GitHub Pages. `CNAME` sets the custom domain `estatemark.co.in`. Do not change it without maintainer approval.
- Keep edits minimal: update HTML/CSS in-place. Do not introduce build tooling or package manifests without discussion.
